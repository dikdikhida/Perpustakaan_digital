## Tentang Project Ini

Project ini dibuat untuk memenuhi tugas mata kuliah Struktur Data. 
Semua fitur CRUD, Searching, Sorting, dan Antrian dibangun dari 0 menggunakan Python tanpa library struktur data eksternal.

**Highlight UI:** 
1.  Background `Kuning Solid #FFD700` biar kontras maksimal.
2.  Font `Hitam Tebal #000` biar kebaca semua.
3.  Semua `input ID, Judul, Tahun` kosong pas pertama dibuka.

### Demo Fitur

|        Fitur       |           Struktur Data         |                   Keterangan                   |
1.  Kelola Buku CRUD |       `Single Linked List`      | Tambah, Tampil, Edit, Hapus, Import/Export CSV |
2.  Peminjaman       |          `Queue FIFO`           |      Sistem antrian sesuai urutan datang       |
3.  Pengembalian     |      `Double Linked List`       |            Riwayat bisa maju-mundur            |
4.  Searching        |    `Linear & Binary Search`     |                Cari buku by Judul              |
5.  Sorting          |`Bubble, Selection, Merge, Quick`|                Urutkan by Judul                |
6.  Index Cepat      |       `Binary Search Tree`      |               Cari by ID O(log n)              |
